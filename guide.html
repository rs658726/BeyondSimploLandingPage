Tasks

General information

The main interface revolves mainly around task management.

Tasks are assigned to custom lists. If a newly created task is not explicitely assigned to a list, it will automatically get assigned to the list designated as the default list. The default list can be selected in Settings.

The Today and Week sections are automatically assigned tasks due today and within the next 7 days, respectively. Use those two sections to plan out your day and your week.

How to create a list?

To create a list, tap the New list button situated at the bottom of the main screen's table. You will be prompted to enter a title for your list and you can also assign your newly created list to a folder. Note that you cannot create two lists with the same title.

How to create a folder?

To create a folder, swipe left on a list which will embed that list in the newly created folder, or simply tap the icon in the configuration bar when creating a new list. Note that you cannot create two folders with the same name.

How to delete a list?

Swipe left on the list you want to delete, and tap the (red) delete button . You will be prompted to confirm. Deleting a list also deletes all of its associated tasks.

How to change the title of a list?

Swipe left on the list you want to edit, and tap the (yellow) edit button.

How to reorder my lists/folders?

Tap and hold the list, note, or folder you want to move until it pops out. Then simply drag it and drop it at the desired location. Note that notes have to be inside a folder on the main screen.

How to sort a list?

You can sort a list by due date/priority in ascending or descending order. To do so, open the list you wish to sort and tap the button in the top right corner of the screen, and choose between ascending and descending sorting in the action sheet.

How to create a task?

You can create a task by tapping the blue button in the bottom right corner of the screen inside the main screen, the Today and Week sections, or inside a custom list. Inside the habit and the notes sections, the button is used to respectively create a new habit and a new note.

Adding a task inside a list will automatically assign the task to the list.

Adding a task inside the Today or Week sections will automatically set the due date to today's date.

Inside the Today and Week sections, you can also add a task by tapping and holding on a section header. This will assign the task to the selected section header by pre-assigning to it a date.

The next two sections explain how to configure a task.

How to configure a task using natural language?

One of the most exciting features of Beyond is the ability to quickly create a task using natural language. You can type in a due date, a time, or a recurrence, and Beyond will highlight it and interpret it. Refer to the table below for a list of examples.

Adding a task inside a list will automatically assign the task to the list.

How to configure a task using the configuration bar?

Your configuration bar shows 4 icons besides the submit button. From left to right, the list selection icon, the scheduler icon, the reminder icon, and the priority level icon.

The list selection icon lets you assign the task to a list.

The scheduler icon allows you to schedule a date, a time, and a recurrence.

Tap the reminder icon to schedule a notification. You can alternatively tap and hold the reminder icon to automatically schedule a reminder for a dated task at the time of the event.

The priority level icon lets you specify a level of priority for a task. There are three different levels of priority each with a distinct color. The green color represents the lowest priority or priority 1. The yellow color represents a medium level of priority or priority 2. Finally, the red color represents the highest level of priority or priority 3.

How to add a task with SIRI?

You can use SIRI to add tasks but there are limitations. For example, SIRI will assign a time by default if being asked to remind you on a date without specifying the time. If a list is not specified in the voice command, Siri will assign the task to the default list (i.e., the list rendered as the default list in settings). On the other hand, if the list spoken in the voice command does not exist, the task won't get added to Beyond.

How to edit a task?

Tap the task you wish to edit. You can also use the edit view to add notes to your tasks.

How to edit multiple tasks?

Inside a list, the Today section, or the Week section, tap the button in the top right corner of the screen, and select Edit Multiple Tasks in the action sheet. Tap the tasks you wish to edit -- they will get highlighted. You have the option to assign to a list, complete, reschedule, set a level of priority, or delete the selected tasks.

How to complete a task?

Simply tap the checkbox associated with your task.

How to uncomplete a task?

Once a task is marked off, an undo button will pop-up at the left bottom side of your screen and you will have 5 seconds to tap it to restore the completed task.

How to reschedule a task?

You can reschedule a task two ways. One is by swipping left on the task which will display a calendar pop-up view.

Alternatively, inside the Today or the Week section, tap and hold the task until it pops out, then drag and drop it to change its due date.

How to search a task?

In the main screen, tap the icon situated in the top right corner, a list of all the existing tasks will appear. Simply type in the relevant key words to search a task.

Habits

General information

The Habits section allows the user to view a list of the habits due today and a list of all habits. Simply use the tab buttons or swipe right/left to switch between the two views.

How to create a habit?

Create a habit by tapping the blue button in the bottom right corner of the screen inside the Habits section.

How to configure a habit?

Configure your habit to fit your needs. Your can specify a frequency (daily, weekly, intervals), a completion number (or the number of times to check-in a habit before it's considered complete), a start date, and a notification time (reminder).

How to view your progress?

In the main Habits section, tap a habit to view your progress on a calendar along with some statistics including streaks and completion rate.

How to edit a habit?

To edit a habit, tap on it. Then, in the detailed view that pops up, tap the Edit button situated in the top right corner of the screen.

How to delete a habit?

To delete a habit, go to the edit view (refer to previous section) and tap delete. You will be prompted to confirm, select delete in the action sheet.

How to complete a habit?

Tap the circle box to complete a habit. If you specified a completion number greater than one, you will need to tap the circle box multiple times.

Alternatively, you can mark off a habit in the calendar view which is accessible by tapping the habit you wish to edit.

How to uncomplete a habit?

Tap the circle box to uncomplete a habit. You will be prompted to confirm, select undo in the action sheet.

Alternatively, you can uncomplete a habit in the calendar view which is accessible by tapping the habit you wish to edit.

Notes

General information

Notes are available inside the Notes section and are ordered by date (last edition).

How to create a note?

Create a note by tapping the blue button in the bottom right corner of the screen inside the Notes section.

How to add a note to a folder?

Open the note you want to assign to a folder and tap the add folder button in the top right corner of the screen.

How to save a note?

You don't need to worry about saving your notes, they are automatically saved at every edit.

How to delete a note?

Swipe left on the note you wish to delete, and tap the (red) delete button .

How to archive a note?

Swipe left on the note you wish to archive, and tap the (blue) archive button .

How to unarchive a note?

To access archived notes, go to Settings by tapping the icon in the top left corner of the main screen, and select Archived Notes. Swipe left on the note you wish to unarchive, and tap the (blue) unarchive button .

How to pin/upin a note?

Swipe left on a note and tap the pin note button. To unpin, follow the same procedure.

How to sort notes?

Notes are sorted automatically by last edition.

How to search a note?

In the Notes section, simply swipe down on your screen to show the search bar, and type in your keywords.
